# @f0c1s/easy-cors

Setup CORS easily

```bash

npm install @f0c1s/easy-cors

```

## Usage

```javascript

import {easyCors} from '@f0c1s/easy-cors';
// expressApp = express();
easyCors(expressApp);

```

Check [this service](https://github.com/localserve/node-cpu-service) for implementation example.
